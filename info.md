# Stromkosten Rechner

Berechnet deine Stromkosten und Solar-Einsparungen in Home Assistant.

## Features

- ✅ Addiert alle 3 Phasen des Shelly 3EM
- ✅ Bis zu 4 Hoymiles Wechselrichter
- ✅ Tägliche Kosten berechnen
- ✅ Solar-Einsparungen tracken
- ✅ Automatischer Reset um Mitternacht

## Installation

1. Füge dieses Repository zu HACS hinzu
2. Suche nach "Stromkosten Rechner"
3. Installiere die Integration
4. Starte Home Assistant neu
5. Füge die Integration hinzu: Einstellungen → Geräte & Dienste

## Konfiguration

Du benötigst:
- 3x Entity IDs vom Shelly 3EM (eine pro Phase)
- 1-4x Entity IDs von Hoymiles (täglicher Ertrag)
- Deinen Strompreis in EUR/kWh
